Flightjar

A live ADS-B map + logger, licensed under the GNU General Public License v3.0.

Runtime libraries

  • Leaflet — BSD-2-Clause. Map engine + controls.
  • FastAPI — MIT. HTTP / WebSocket framework.
  • uvicorn — BSD-3-Clause. ASGI server.
  • pyModeS — GPL-3.0-or-later. Mode S / ADS-B decoder.
  • httpx — BSD-3-Clause. Async HTTP client (adsbdb calls).

Data & assets

  • tar1090-db / Mictronics — aircraft registry (ICAO24 → registration / type). CC0.
  • tar1090 — per-type aircraft silhouettes (markers.js). GPL-2.0+.
  • OurAirports — global airport database + coordinates. Public domain.
  • OpenStreetMap contributors — base map tiles. ODbL.
  • CARTO — Dark base tiles (on top of OSM data).
  • Esri — satellite imagery tiles.
  • adsbdb.com — origin / destination by callsign, plus per-tail details (operator, country of registration, photo URL).
  • airport-data.com — aircraft photographs shown in the detail panel, surfaced via adsbdb.
  • flagcdn.com — country flag SVGs shown in the sidebar and detail panel.
