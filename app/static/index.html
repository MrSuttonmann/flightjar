    Flightjar

    A live ADS-B map + logger, licensed under the GNU General Public License v3.0.

    GitHub

    Runtime libraries

    • Leaflet — BSD-2-Clause. Map engine + controls.
    • Lucide — ISC. Icon set used throughout the UI (weather overlay, footer controls, dialog buttons).
    • ASP.NET Core — MIT. HTTP / WebSocket framework and background-service host.
    • pyModeS — MIT. Mode S / ADS-B decoding algorithms — hand-ported to C# from pyModeS 3.2.0, with the original reference kept on each ported module.

    Data & assets

    • tar1090-db / Mictronics — aircraft registry (ICAO24 → registration / type). CC0.
    • tar1090 — per-type aircraft silhouettes (markers.js). GPL-2.0+.
    • OurAirports — global airport + navaid database (coords, types, frequencies). Public domain.
    • OpenStreetMap contributors — base map tiles. ODbL.
    • CARTO — Dark base tiles (on top of OSM data).
    • Esri — satellite imagery tiles.
    • adsbdb.com — origin / destination by callsign, plus per-tail details (operator, country of registration, photo URL).
    • planespotters.net — primary source of the aircraft photographs shown in the detail panel (photographer credit shown on each photo).
    • airport-data.com — fallback photo source when planespotters has nothing for a tail, surfaced via adsbdb.
    • flagcdn.com — country flag SVGs shown in the sidebar and detail panel.
    • OpenFlights — airline database (IATA + ICAO codes, callsigns). ODbL.
    • aviationweather.gov — NOAA-published METARs shown alongside origin / destination airports on the detail panel. Public domain.
    • OpenAIP — worldwide aeronautical chart tiles (airspaces, navaids, obstacles) for the optional Aeronautical overlay. CC BY-NC-SA, requires a personal API key.
    • VFRMap.com — FAA IFR Low / IFR High enroute tiles for the optional US-only chart overlays. Charts are US Federal Government public domain; tile hosting by VFRMap.com.
